Instructions For Creating a Webpage on Github

1. Create an account on Github.com.

2. Create a new repository with the name "your user name.github.io".

3. Add an HTML file that includes any coding needed by uploading or using drag and drop.

4. Upload any other files needed such as pictures the same way.

5. Click commit changes to save and build the project.

6. Check your webpage to be sure it works.
